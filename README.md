# Expense Calculator

*  A simple web-based application to track your daily expenses.
*  You can add amounts, categorize them, filter them, and reset all data.
*  This project demonstrates basic HTML, CSS, and JavaScript (CRUD operations).

# Features

*  Add new expenses (Create)
*  View all expenses (Read)
*  Edit existing expenses (Update)
*  Delete expenses (Delete)
*  Reset the entries 
*  Clean and simple UI

# CRUD Explanation

This project uses all CRUD operations:

**Operation** ---	**Meaning**	 ---   **In This App**

Create	-         Add new data	 -     Adding a new expense/income

Read	  -       View existing data	-  Listing all expenses/income

Update	 -        Modify data	 -     Editing an expense/income

Delete	  -       Remove data	  -    Deleting an expense/income

# How to Use

* Select a category(income/expense)
* Enter an amount
* Click Add Expense
* filter buttons to view specific categories(ALL,Income,Expense)
* Edit or delete items using action buttons
* Click Reset to clear all entries
